# Quick Setup Guide

## Step 1: Run Reorganization
```bash
scripts/REORGANIZE.bat
```

## Step 2: Install Tools
```bash
pip install uv
uv sync
```

## Step 3: Commit Changes
```bash
git add .
git commit -m "refactor: Reorganize repository structure"
git push
```

## Done! 

Your repository is now organized and modern.
